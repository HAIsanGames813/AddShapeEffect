# 図形貼り付け
## 概要
図形をエフェクトとして追加するプラグインです

## ダウンロード方法
[ここからダウンロード](https://github.com/HAIsanGames813/AddShapeEffect/releases)

## 使用方法
映像エフェクトの装飾タブに「図形貼り付け」が追加されます。  
パラメーターなどの解説は[YMM4情報サイト](https://ymm4-info.net/ymme/%E5%9B%B3%E5%BD%A2%E8%B2%BC%E3%82%8A%E4%BB%98%E3%81%91)にてご確認ください。  
※exo出力には対応していません
